# Redis Connection Pool Test
Testing Python GIL &amp; uWSGI &amp; Redis connection pool

## Installation

- Python 2.7
- uWSGI 2.0.10
- Redis 3.0

## Python modules

- bottle
- redis
- gevent

## Run

- Install and start uWSGI using `default-uwsgi.ini` as configuration file.
- Install and start Redis.
- Do some testing.
